Mode d'emploi:
Allez dans le dossier qui contient le code. Le compilateur par défaut est gcc, donc il faut vérifier que vous l'avez installé.

Pour compiler:
Tapez 'make' dans le terminal et puis tapez entrée.

Pour exécuter le programme compilé:
Tapez './AirWatcher' dans le terminal et puis tapez entrée.

Pour choisir une fonctionnalité:
Tapez l'indice indiqué au début de chaque fonctionnalité dans le terminal et puis tapez entrée. S'il faut saisir des paramètres supplémentaires, veuillez suivre les indications.
